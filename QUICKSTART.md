# 快速启动指南

## 1. 安装依赖

```bash
cd frontend
npm install
```

## 2. 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:3000` 启动

## 3. 登录测试

### 普通用户
- 用户名: `user`
- 密码: `123456`

### 超级管理员
- 用户名: `admin`
- 密码: `123456`

## 4. 功能说明

### 普通用户功能
- **仪表盘**: 查看系统统计数据和快速操作
- **任务管理**: 创建、启动、停止、删除任务
- **数据管理**: 查看、编辑、删除、导入导出数据
- **个人中心**: 修改个人信息和密码

### 超级管理员功能
除了普通用户的所有功能外，还可以：
- **用户管理**: 创建、编辑、删除、启用/禁用用户
- **系统设置**: 配置系统参数、数据库设置、系统操作
- **操作日志**: 查看所有用户的操作记录

## 5. 接口对接

所有API接口定义在 `src/api/` 目录下，目前为占位实现。需要对接实际后端时：

1. 修改 `.env.development` 和 `.env.production` 中的 `VITE_API_BASE_URL`
2. 在对应的API文件中替换接口地址和数据处理逻辑
3. 根据后端返回格式调整 `src/api/request.ts` 中的响应拦截器

## 6. 构建生产版本

```bash
npm run build
```

构建产物在 `dist` 目录

## 7. 预览生产构建

```bash
npm run preview
```

## 注意事项

- 当前登录功能使用 localStorage 模拟，实际应使用后端返回的 token
- 所有数据为示例数据，需要对接真实后端接口
- 图表功能需要集成 ECharts 或其他图表库
- 文件上传/下载功能需要根据实际需求实现
